坚持锻炼，坚持早起

发表于 2018-06-07

这篇文章对前一周进行总结，看看之前立的flag倒了多少。

回顾过去的一周

  1. 嵌入式课设坚决不躺着
    这一条基本是完成了吧，自己完成QT部分的内容，课设也顺利验完了。
  2. 坚持锻炼，每周至少跑步三次
    这个做得不是很好，但是现在看来也未可知，毕竟今天才周四，今天跑步。
  3. 坚持早起，六点半起床
    这一点做的很好，除了有一天晚上睡的比较晚，九点起床之外，基本都是六点半就起了，继续坚持。
  4. 关于夏令营
    真的就没有再投，哈哈哈，实在没心思关注这个了，关于同济网站提交的时候，只能怪校网了，哈圾哈圾。

接下来做什么

  1. 下周要进行数据库课程设计了，软件工程全程躺过来的，网站还都不会搭，从头开始学，emmm，九月份之前怎么也得搞一个稍微好点的项目吧，不然面试真的很尬。

  2. 运动会的时候看到同学在复习代几，真的很厉害，我还什么都没看。。。现在是6月7日，下周还有软件工程考试，emmm，关键还不知道怎么复习，算了，先复习再说。

  3. 下周开始数据库课程设计，我是自己一组？感觉还是蛮虚的，找谁组队呢？这是个问题，时间应该挺紧的。估计也没有时间复习啊，现在是15周，做完课设就到了17周结束，要是能进同济夏令营的话就还有两周的实时间复习，计算机组成原理、操作系统、计算机网络、数据结构，好像来不太及哎，蓝瘦。

  4. 坚持健身，至少每周三次健身房和跑步。
